# n8n-Project
This n8n workflow is an AI-powered email automation system designed to save time by automatically reading, understanding, categorizing, and replying to incoming Gmail messages. It acts like a smart virtual assistant for handling business communication.

AI-Powered Email Automation System using n8n & Gemini AI

Built an intelligent email automation workflow in n8n that automatically monitors incoming Gmail messages, analyzes their content using Google Gemini AI, categorizes emails based on intent, and sends personalized responses without manual intervention.

The system extracts sender information, generates dynamic greetings, classifies emails into categories such as course inquiries, consulting requests, payment-related discussions, college communications, and miscellaneous queries, then applies Gmail labels and sends tailored auto-replies accordingly.

Key features include:

Real-time Gmail monitoring for unread emails
AI-based sender name extraction using Gemini AI
Smart email classification with custom business categories
Personalized response generation based on email intent
Automated Gmail labeling for organized inbox management
Workflow branching and conditional logic handling in n8n
Reduced manual effort in handling repetitive email communication

This automation helps streamline client communication, improve response time, and maintain professional engagement at scale.
